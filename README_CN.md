## ArtNex: Tech, Art, Tomorrow! 🎆

ArtNex是一个的深度学习框架，其使命是推动艺术和科技的融合，致力于开创下一代创意表达和技术创新的可能性。该框架具有高度灵活的架构，旨在为艺术家、设计师和工程师提供一个强大而创新的工具，以探索前沿的艺术形式和科技应用。

## 安装

要安装项目，请使用以下命令：

```python
pip install artnex
```
